## Personal websites

[Maggie Appleton](https://maggieappleton.com/) - her site inspired me to start my own 'digital garden'.

[Gwern](https://gwern.net/index) - probably biggest content-wise personal site I ever saw.

[100 Rabbits](https://100r.co/site/home.html) - couple living permanently on a boat, making minimalistic software and (not-so-minimalistic) art.

[Vajra Chandrasekera](https://vajra.me/) - private website of Sri Lankan speculative fiction writer, thoughtful posts about social reality of Sri Lanka on his blog.

[Baldur Bjarnason](https://www.baldurbjarnason.com/) - deep cutting tech criticism.

[Emily Gorcenski](https://emilygorcenski.com/) - mathematics, tech, politics. Her website is beautiful in very subtle way.

[The Marginalian](https://www.themarginalian.org/) - long running website, assortment of curiosities.

[The Folkmoss Log](https://folkmoss.bearblog.dev/) - personal website structured and stylized like an old-school book.

[Rishikesh Sreehari](https://rishikeshs.com/) - diverse, nicheless content, additionally malayali perspective.

[moonlight computer](https://moonlight.computer/) - cool retro vibes.

[ribo.zone](https://ribo.zone/) - biology, bugs, mecha. Very original design, rough, messy, somehow old-school, but also fresh.

[localghost](https://localghost.dev/) - old school style website with multiple "skins".

[Shellsharks](https://shellsharks.com/) - indie web, web security.

[anhvn](https://anhvn.com/) - art, vivid beautiful design.

[ava's space](https://blog.avas.space/) - cute, minimalistic design.

[Robb Knight](https://rknight.me/) - indie web, a lot of inspiration for website and pointers to others.

[James' Coffee Blog](https://jamesg.blog/) - lovely and minimalistic.

[Manuel Moreale](https://manuelmoreale.com/) - full of interviews with other web creators.

[Konfetti Explorations](https://marisabel.nl/) - beauty, joy, quirky, really personal

[Josh W. Comeau](https://www.joshwcomeau.com/) - web animations, frontend

[Everest Pipkin](https://everest-pipkin.com/) - assortment of one person web projects and minigames

[A Working Library](https://aworkinglibrary.com/) - reading, writing, book collection, tech criticism. She has really nice format for book reviews.

[Zachary Kai](https://zacharykai.net/) - simple design, a lot of pointers and links for inspirations.

[Robin Rendle](https://robinrendle.com/) - unusual presentation in style of documentation website.

[Nikhil Anand](https://nikhil.io/) - cool style, logs, reviews.

[Joel's Homepage](https://joelchrono.xyz/) - video games, ttrpg, science fiction.

[Lost Letters](https://lostletters.neocities.org/) - cutesy, old-school girlie website, geocities/neocities vibe.

[Henry From Online](https://henry.codes/) - probably one of the most beautiful personal site around, but still (mostly) monochromatic.

[Bino](https://binovarghese.com/) - software development, travel, writing in malayalam

[Molly White](https://www.mollywhite.net/) - prominent tech critic, started with criticizing crypto.

[Pluralistic](https://pluralistic.net/) - blog by Cory Doctorow himself, old time blogger, tech critic and science fiction writer.

## Magazines

[Quanta Magazine](https://www.quantamagazine.org/) - popular science with beautiful, beautiful illustrations. Often touches frontiers, new stuff. It is approachable for an amateur, but doesn't shy away from going slightly deeper.

[Asterisk](https://asteriskmag.com/) - curious assortment of diverse, interesting essays. It can be biased towards (right) libertarian and Sillicon Valley aligned worldviews though.

## Web Dev Resources

[32-Bit Cafe](https://32bit.cafe/) - plenty of resources for creating websites.

[CSS Tricks](https://css-tricks.com/) - essential for front end, css focused.

[Whimsica11y](https://whimsica11y.net/) - whimsical accessibility frontend
